BENEFITS
================================================================================

Clear Separation
- Production vs Research code clearly separated
- Official pipeline ready for publication
- Experimental work isolated for development

Scalable Architecture
- Sequential pipeline (step_0 -> step_18)
- Modular design - easy to extend
- Research-friendly - supports experimentation

Workflow
EXPLORATORY -> OFFICIAL PIPELINE -> PUBLICATION
(Research)   (Validation)        (Manuscript)
